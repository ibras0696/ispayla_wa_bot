# Чек‑лист готовности (бот WhatsApp)

Обязательное для текущего релиза:
- [x] Меню «Профиль/Продажа/Покупка», кнопка «Назад» из профиля и «Моих объявлений».
- [x] Мастер продажи с полями: заголовок, описание, цена, марка, модель, год, пробег, VIN, регион, состояние; до 3 фото; отмена «отмена»/«меню» с подсказкой.
- [x] Карточка своих объявлений: регион/состояние/модель, до 3 фото.
- [x] Каталог/поиск/фильтры/избранное: выбор по номеру/ID, карточка выдаёт до 3 фото и контакты.
- [x] Тексты меню и подсказок упрощены.
- [x] Alembic ревизия для полей `model_name`, `region`, `condition`.
- [ ] Прогнать миграцию в контейнере (`docker compose exec bot alembic upgrade head`).
- [ ] Проверить, что файлы фото реально лежат в `media/uploads` контейнера (иначе карточки придут без части фото).

Отложено/следующие шаги:
- [ ] Кнопки «Позвонить/WhatsApp» в карточке каталога (сейчас текстовая ссылка).
- [x] Фильтры по региону/состоянию.
- [ ] Состояния в Redis вместо памяти/файла.
- [ ] Админ/модерация/статусы объявлений.
- [ ] Тесты и CI, миграции продолжать фиксировать через Alembic.
